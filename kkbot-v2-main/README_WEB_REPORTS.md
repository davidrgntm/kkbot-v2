# Web Hisobotlar

Bu versiyada botdagi 🧾 Hisobotlar web panelga qo‘shildi.

Yo‘l:
- /reports — oy va filial tanlash
- /reports/download — Excel hisobot yuklab olish
- /export — eski tabel sahifasi o‘rniga /reports ga yo‘naltiriladi

Hisobot botdagi formatga o‘xshash:
- ФИО
- Должность
- c / до / итого
- oy kunlari
- 1-15
- 16-end
- JAMI

Manba: SQLite shifts jadvali. Google Sheets export/import logikasiga tegilmadi.
